# Design Tokens

Generate Design Tokens based on Figma file.

Edit config obj in buildDictionary.js for more plattform support

<br>

Build all files:
```bash
npm run dev
```

<br>

Generate Tokens:
```bash
npm run generate
```

<br>

Build Dictionary Files:
```bash
npm run build-dictionary
```

<br>
